Radiosity engine
================
About
-----
This is a simple radiosity engine written in C++ using Qt.

Usage: 
```
radiosity-engine.exe --scene=scene.xml --output=image.png --resolution_x=512 --resolution_y=512 --num_iterations=10 --patch_size=1.5 --num_rays_per_patch=5000
```
